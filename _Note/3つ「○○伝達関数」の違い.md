# 3つ「○○伝達関数」の違い
- 開ループ伝達関数
- 一巡伝達関数
- 閉ループ伝達関数

| 名称       | 数式                                            | 意味                                               |
| -------- | --------------------------------------------- | ------------------------------------------------ |
| 開ループ伝達関数 | $C(s)P(s)$                                    | ループを切り開いた状態の，入力→出力の一方向の伝達関数．フィードバック要素 H(s) を含まない |
| 一巡伝達関数   | $C(s)P(s)H(s)$                                | ループを切り開いた上で，フィードバック要素も含めてループを一周したときの伝達関数         |
| 閉ループ伝達関数 | $$<br>\frac{C(s)P(s)}{1 + C(s)P(s)H(s)}<br>$$ | ループを閉じた状態での，目標値→出力の伝達関数                          |
★「ユニティフィードバック」（ $H(s) = 1$ ）のとき，  
→開ループ伝達関数 と 一巡伝達関数 は一致する
