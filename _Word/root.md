# 根（Root）
> 特性方程式（伝達関数の分母多項式=0とした方程式）を満たす複素数 s の値．

一般に，多項式がゼロとなるような方程式の解を根（あるいは零点）と呼ぶ．  
制御工学では，「根 = 特性方程式の根 = 閉ループ系の極」 となるため，ほぼ「根 = 極」  
- 根の実部の符号　→　安定性（左半平面ならば安定）
- 根の虚部の　→　新動特性
